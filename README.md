Code for Omnichannel Retailing under partial lost-sales. Please see tests/DES for examples of how to run the discrete event system. See SimStudy/sim_study_template_plt.py for running the positive lead time study in the paper, and SimStudy/sim_study_template_zlt.py for the zero lead time equivalent.

This is open source version of the codebase used for the paper: *Periodic review inventory control for an omnichannel retailer with partial lost-sales (2025)* by Lowery, Sachs, Eckley and Lloyd
